<div align="center">
  <strong>linkerland</strong> is a tool to parse and visualize linker map files generated with clang lld, GNU ld. 
</div>


## Usage

### Command Line

```shell
linkerland <linker_map_file>
```

### GUI

```shell
linkerland --ui <linker_map_file>
```
