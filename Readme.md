**linkerland** is a tool to parse and visualize linker map files. 

> linkerland is still *heavily* work-in-progress.

## Usage

### Command Line

```shell
linkerland <linker_map_file>
```

### GUI

```shell
linkerland --ui <linker_map_file>
```
