## What is linkerland?

**linkerland** is a tool to parse and visualize linker map files. linkerland is still work-in-progress.

## Usage

### Command Line

```shell
linkerland <linker_map_file>
```

### GUI

```shell
linkerland --ui <linker_map_file>
```
